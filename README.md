# Task 2 – Exploratory Data Analysis (EDA)

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a dataset to understand data patterns, relationships, and key insights. EDA helps summarize the main characteristics of the dataset using statistics and visualizations.

## Objectives
- Understand the dataset structure
- Identify missing values and data types
- Perform statistical analysis
- Visualize relationships between variables
- Extract meaningful insights from the data

##  Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Dataset
The dataset is analyzed to explore patterns and relationships between different features. Basic preprocessing steps such as handling missing values and checking duplicates are performed before analysis.

##  Steps Performed
1. Import libraries
2. Load dataset
3. Understand dataset structure
4. Handle missing values
5. Perform statistical summary
6. Univariate analysis
7. Bivariate analysis
8. Correlation analysis
9. Data visualization

##  Visualizations Used
- Histograms
- Count plots
- Box plots
- Scatter plots
- Correlation heatmap

##  Key Insights
- Identified patterns and trends in the dataset
- Detected missing values and outliers
- Found correlations between important variables

##  How to Run
1. Clone the repository
2. Install dependencies
pip install pandas numpy matplotlib seaborn
3. Open the Jupyter Notebook and run all cells.

##  Conclusion
EDA helps in understanding the dataset and preparing it for further machine learning or predictive modeling.
